# zsh-git-alias
Extracted git aliases from https://github.com/robbyrussell/oh-my-zsh.

If you use the non-recommended on-my-zsh, you can install this directly to your ~/.bash_profile:



### Usage

```shell
curl -s https://raw.githubusercontent.com/a6051529/zsh-git-alias/master/git.plugin.zsh >>
~/.bash_profile
```
